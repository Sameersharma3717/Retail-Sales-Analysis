# 🛒 Retail Sales Analysis

## 📌 Project Overview

This project is an end-to-end Retail Sales Analysis built using **SQL, Python, Excel, and Power BI**. The objective was to analyze sales performance, identify business trends, and create interactive dashboards to support business decision-making.

---

## 🎯 Business Objectives

- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Track monthly sales trends
- Compare profitability across states
- Understand customer segment performance
- Build interactive dashboards for business users

---

## 🛠️ Tools & Technologies

- SQL
- Python (Pandas, Matplotlib)
- Microsoft Excel
- Power BI
- GitHub

---

## 📂 Dataset

- Sample Superstore Dataset
- Records: ~9,994 Orders

---

## 📊 Key Business Insights

- Technology is the highest-performing category by sales.
- Sales peak during Q4, especially in November.
- Consumer customers contribute the largest share of total sales.
- California and New York generate the highest profits.
- Higher discounts are often associated with lower profitability.
